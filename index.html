Premise
Setting Premise

《Hexa Arena》发生在一个由 AI 主导生产、分配与治理的未来社会。基础生存压力已经被大幅削弱，但人类的冒险欲、攻击性、观看欲与自我证明需求并没有消失。于是，竞技场成为一种被制度化、娱乐化、商品化的公开试炼。

Page 01

World
World

Hexa Arena 的世界不是崩坏废土，而是一个过于稳定、过于安全、过于高效的未来社会。AI 负责大部分生产、分配与秩序维护，人类寿命被延长，物质极大丰富，日常生活被优化到近乎没有生存焦虑。

正因为如此，竞技场才获得了新的社会意义。它不是落后的野蛮遗迹，而是这个时代用来疏导风险冲动、表演欲与竞争欲的正式机制。参赛者既是战斗者，也是被观看、被评级、被消费的竞技表演者。

Page 02

Match Start
Match Start

每一局比赛都从战区投送开始。战场建立在六边形棋盘之上，参赛者被投入同一片公开空间，没有固定阵营，只能依靠自己的职业路线、装备构筑和局内判断生存下去。

首发三种职业分别验证三条不同逻辑：战士依靠高刚性、推进与压制建立正面优势；刺客依靠速度、切入与时机制造致命碰撞；电气工程师则用金属结构感的轻型护甲、护盾与能量循环维持技术性压制。

Page 03

Search And Clash
Search And Clash

Hexa Arena 的核心并不是单纯的对打，而是“搜索、判断、碰撞、撤离”的连续过程。参赛者需要在有限信息中移动、搜索资源、评估风险，并决定自己是否继续深入战区。

当两名或更多参赛者进入同一格，距离会被瞬间压缩，局势从试探转为直接冲突。攻击、逃跑、压制与反制都在同一空间内发生，这使得每次接触都具有极高的戏剧张力和结算重量。

Page 04

Extraction And Build
Extraction And Build

在这套设定里，胜利不只等于击败别人。真正重要的是能否把战利品与收益活着带出竞技场。撤离不是失败的替代动作，而是被规则明确承认的一种结果与价值判断。

赛后，角色会重新回到构筑循环。核心躯体决定底层身份与属性倾向，装备决定常驻战力，芯片决定战术路线，药物决定短时爆发，道具决定局势操作。于是每一局既是生死试炼，也是下一次更高等级竞技的准备阶段。
