# Hexa Arena Setting Site

Static site bundle for sharing the `Hexa Arena` setting page as a standalone public repository.

## What this contains

- `index.html`: site entry
- `assets/`: shared art assets
- `.github/workflows/pages.yml`: GitHub Pages deployment workflow

## Recommended publish flow

1. Create a new **public** GitHub repository
   Example: `hexa-arena-setting-site`
2. Copy all files from this directory into that repository
3. Push to the repository's `main` branch
4. In GitHub:
   - Open `Settings > Pages`
   - Set Source to `GitHub Actions`

## Expected result

The site should publish to a URL like:

`https://<your-user>.github.io/<repo-name>/`

## Notes

- This bundle is intentionally separated from the main game repository
- It is suitable for public sharing without exposing the rest of the project workspace
